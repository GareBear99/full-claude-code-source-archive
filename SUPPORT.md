# Support

## What this repository supports

This project is maintained by **Gary Doman / GareBear99** as a public-facing archive, architecture index, and MCP exploration toolkit.

Good support topics:

- repository structure and navigation
- README or docs clarifications
- MCP server setup
- package metadata and release-surface cleanup
- public-facing discoverability improvements

## Best places to ask

- **GitHub Discussions** for setup help, publishing questions, and general repo guidance
- **GitHub Issues** for confirmed bugs or focused feature requests

## What support does not imply

This repository is **not** presented as an official upstream vendor release or an official support channel for Anthropic products.

Support is best-effort and focused on this public archive/research repo itself.
